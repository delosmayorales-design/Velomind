Configurar Strava OAuth

Strava usa OAuth 2.0. Solo necesitas crear una aplicación gratuita en su portal de desarrolladores.

1
Ve a strava.com/settings/api e inicia sesión
2
Crea una nueva aplicación. En "Authorization Callback Domain" pon tu dominio (o localhost para desarrollo)
3
Copia tu Client ID y Client Secret
4
Configura en backend: STRAVA_CLIENT_ID, STRAVA_CLIENT_SECRET y STRAVA_REDIRECT_URI en backend/.env
5
Usa los endpoints backend ya disponibles: /api/providers/strava/connect y /api/providers/strava/callback
Scopes necesarios:
read - Perfil publico del atleta
activity:read_all - Todas las actividades

Configurar Garmin Connect

Garmin usa OAuth 1.0a y requiere aprobación de su Developer Program. El proceso es más complejo pero da acceso a datos muy ricos.

1
Regístrate en Garmin Developer Program (puede tardar unos días en aprobarse)
2
Crea una aplicación y obtén tu Consumer Key y Consumer Secret
3
Instala el paquete npm: npm install garmin-connect en tu backend Node.js
4
Crea rutas backend: /api/auth/garmin/start y /api/auth/garmin/callback
5
Alternativamente, usa la Garmin Health API para acceso simplificado a datos de salud
Datos disponibles:
Actividades con archivos .FIT completos
VO2max, Training Status, HRV
Body Battery, sueño, estrés diario

Ejemplo de backend (Node.js / Express)

// server.js â€” Backend para OAuth Strava
const express = require('express');
const axios = require('axios');
const app = express();

const STRAVA_CLIENT_ID = process.env.STRAVA_CLIENT_ID;
const STRAVA_CLIENT_SECRET = process.env.STRAVA_CLIENT_SECRET;
const REDIRECT_URI = 'https://tudominio.com/oauth-callback.html';

// Intercambio de código por tokens
app.get(window.API_URL + '/auth/strava/callback', async (req, res) => {
  const { code } = req.query;
  const response = await axios.post('https://www.strava.com/oauth/token', {
    client_id: STRAVA_CLIENT_ID,
    client_secret: STRAVA_CLIENT_SECRET,
    code,
    grant_type: 'authorization_code'
  });
  const { access_token, refresh_token, expires_at, athlete } = response.data;
  // Guardar tokens en DB, crear sesión JWT...
  res.json({ access_token, athlete });
});

// Sincronizar actividades
app.get('/api/strava/activities', async (req, res) => {
  const { access_token } = req.user; // from JWT middleware
  const activities = await axios.get('https://www.strava.com/api/v3/athlete/activities', {
    headers: { Authorization: `Bearer ${access_token}` },
    params: { per_page: 50, after: req.query.after }
  });
  res.json(activities.data);
});
